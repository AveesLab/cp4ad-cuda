# [Week 6-7] CUDA

$ git clone https://github.com/AveesLab/cp4ad-cuda.git

### How to Compile
**C**
```
$ gcc -o <excution file name> <source code file name>
```

**CUDA**
```
$ nvcc -o <excution file name> <source code file name>
```

### Excution

```
$ ./<excution file name>
```
